## 简介

## 安装
1. 克隆仓库：
    ```bash
    git clone https://github.com/YeHeng/py.git
    ```
2. 进入项目目录：
    ```bash
    cd py
    ```
3. 安装依赖：
    ```bash
    pip install -r requirements.txt
    ```

## 使用
1. 启动项目：
    ```bash
    python main.py
    ```
